# customer-purchase-data-analysis_R

Exploratory Data Analysis of customer purchase behavior using R.  
Analyzes revenue trends, customer demographics, product categories, and seasonal patterns using a Kaggle dataset.  
Results are documented in R scripts, R Markdown, and a final PDF report.

## Files
- `analysis.R` – R script with all data processing and analysis  
- `analysis.Rmd` – R Markdown file containing code, outputs, and narrative  
- `report.pdf` – Final report generated from R Markdown  

## Dataset
Data sourced from [Kaggle: User Purchase Behavior Analysis Dataset](https://www.kaggle.com/datasets/refiaozturk/online-shopping-dataset)
