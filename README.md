# Customer Purchase Data Analysis

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a customer purchase dataset sourced from Kaggle.  
The goal is to analyze revenue trends, customer demographics, product categories, and seasonal patterns, and provide actionable business insights.

## Dataset
**Source:** [Kaggle: User Purchase Behavior Analysis Dataset](https://www.kaggle.com/datasets/refiaozturk/online-shopping-dataset)  
Contains user transactions with attributes such as User ID, Age, Gender, Country, Purchase Amount, Purchase Date, and Product Category.  
Used for data cleaning, summarization, visualization, and analysis in this project.

## Tools Used
- **R** – Data manipulation, analysis, and visualization  
- **R Markdown** – Code documentation and report generation  
- **PDF** – Final project report

## Analysis & Key Findings
- **Top Country:** France generated the highest total revenue (18%), mainly driven by the Clothing category.  
- **Mid-tier Countries:** Canada, Germany, USA, and Australia (~16% each)  
- **Top Product Category:** Clothing (~$400K), with other categories showing moderately similar revenues.  
- **Customer Demographics:**  
  - Highest spending: Age 35–44  
  - Revenue relatively balanced across genders  
- **Seasonal Trends:**  
  - Revenue peaks: August & year-end (Sep–Dec)  
  - Revenue troughs: February, April, November  
- **Daily Revenue:** Relatively stable across weekdays

## Report & Visualization
- **Customer_Purchase_Analysis.R** → R script with all analysis  
- **Customer_Purchase_Analysis.Rmd** → R Markdown with code, outputs, and narrative  
- **Customer_Purchase_Analysis.pdf** → Final PDF report with visualizations and insights

> **Note:** PDF can be downloaded to view charts and detailed analysis.

## Recommendations
- Strengthen France’s Clothing segment  
- Explore growth in mid-tier markets  
- Promote low-performing categories like Beauty  
- Prepare inventory/marketing for peak months and promotions for low months  
- Target high-spending age group (35–44) while maintaining inclusive marketing across genders
